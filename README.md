# phone-marker

`phone-marker` 是一个纯浏览器运行的图片时间补丁工具，可在本地为手机截图或其他图片添加可调整的时间水印。

所有图片处理均在浏览器本地完成，不会上传到服务器。

当前版本：`v1.0.3`

## 功能

- 批量导入图片
- 添加小时与分钟时间补丁
- 自动补齐两位数字
- 调整字体、字号、颜色和背景颜色
- 使用吸管直接拾取文字或背景颜色
- 调整数字间距、数字宽度和冒号上下偏移
- 调整背景大小、圆角和透明度
- 调整补丁不透明度与缩放大小
- 25%、50%、75%、100% 画布缩放预设
- 自动读取 `fonts` 文件夹中的本地字体
- 导出当前图片或批量导出

## 快速开始

### 推荐方式

在 Windows 上运行：

```powershell
.\启动-phone-marker.ps1
```

脚本会启动本地 HTTP 服务，并自动打开：

```text
http://127.0.0.1:8765/phone-marker.html
```

关闭脚本窗口或按 `Ctrl+C` 即可停止服务。

### 手动启动

项目目录内执行：

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8765/phone-marker.html
```

## 使用方法

1. 导入需要处理的图片。
2. 设置时间、字体、字号和颜色。
3. 点击黄色的“添加时间补丁”按钮。
4. 在预览画布中拖动时间补丁调整位置。
5. 调整补丁大小、透明度和背景样式。
6. 导出当前图片或批量导出全部图片。

## 数字宽度

控制面板提供“数字宽度”滑杆，类似 Photoshop 字符面板里的横向缩放：

- 范围：`50%` 到 `150%`
- 默认值：`100%`
- 小于 `100%` 时数字会横向压缩
- 大于 `100%` 时数字会横向拉伸

该设置会同时影响预览画布和导出结果。新增水印、选中已有水印、调整滑杆时，数字宽度都会保存并实时刷新。

如果截图里的系统时间数字偏窄或偏宽，可以配合字体、字号、数字间距和冒号偏移一起微调，让补丁更接近原图。

## 字体

程序启动时会自动扫描 `fonts` 目录，支持以下格式：

- TTF
- OTF
- WOFF
- WOFF2
- TTC

添加新字体后，刷新页面即可在字体下拉框中选择。

默认字体为 `Google Sans Regular`，默认字号为 `56px`。

## 运行要求

- Windows 10 或 Windows 11
- Python 3
- Chromium 内核浏览器，例如 Chrome 或 Edge

吸管功能依赖浏览器的 EyeDropper API。若浏览器不支持，将自动打开普通颜色选择器。
